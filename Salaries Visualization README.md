# Data Visualization with Python

This repository contains Python code for visualizing a dataset using popular libraries such as Pandas, Seaborn, and Matplotlib.

## Dataset

The dataset used in this project is named `salaries.csv`. It contains information about companies, job roles, degree qualifications, and whether the salary exceeds $100k.

## Code

The Python code (`visualization.py`) reads the dataset and generates visualizations to explore the data. The following visualizations are created:

1. Count of Salary More than $100k
2. Count of Salary More than $100k by Company
3. Count of Salary More than $100k by Job
4. Count of Salary More than $100k by Degree

## Instructions

To use this code, follow these steps:

1. Clone the repository to your local machine.
2. Ensure you have Python installed.
3. Install the required libraries: Pandas, Seaborn, and Matplotlib.
4. Run the Python script `visualization.py`.
